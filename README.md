# server-logger

Node.js server logger with web based terminal

## Getting started

```shell
$ git clone git@github.com:arvindr21/server-logger.git
$ cd server-logger
$ cd server && npm install
$ cd ../client && npm install
```

## Run the app

Start Server

```shell
$ cd server
$ node app.js
```

Navigate to http://localhost:3000 to view the logger console.

Start Client

```shell
$ cd client
$ node index.js
```
